`dotnet ef` がインストールされていない場合は、グローバル ツールとしてインストールします。

```dotnetcli
  dotnet tool install --global dotnet-ef
```

EF Core 用の CLI の詳細については、[.Net CLI の EF Core ツール リファレンス](/ef/core/miscellaneous/cli/dotnet)を参照してください。
